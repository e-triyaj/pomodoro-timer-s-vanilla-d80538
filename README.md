# Pomodoro Timer S Vanilla

Pomodoro Timer S Vanilla

A focus timer with classic Pomodoro intervals. The user starts a 25-minute work session by clicking Start. After 25 minutes the timer announces a 5-minute break with an audible chime; after every 4 work sessions the break is 15 minutes instead. The current phase (work / short break / long break) and the session number are shown clearly. Buttons: Start, Pause, Reset. The page remembers the last 5 completed sessions in localStorage and shows them in a small history panel that the user can clear with one button. A theme toggle switches between light and dark, persisted in localStorage. The page is keyboard friendly: Space starts/pauses, R resets.

## Adding Your Logo

The app ships with a placeholder monogram (the letters **PTS**
in your primary color, `#f00ad5`). To use your own logo:

1. **Prepare your logo.** SVG is preferred (vector, scales cleanly).
   PNG with transparent background is fine — keep it at least
   256×256 px. Maximum 2 MB.
2. **Drop the file** at `frontend/public/assets/brand/logo.svg` (overwrite the placeholder).
   The path is hard-coded — don't rename the file.
3. **Refresh the page.** The header and any logo-using component
   pick up the new file automatically. No code changes needed.

Optional: add `logo-dark.svg` in the same folder for dark theme.
